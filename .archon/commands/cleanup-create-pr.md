# Create Cleanup PR

**Workflow ID**: $WORKFLOW_ID

---

## Your Mission

Create a GitHub draft PR for this cleanup work package.

---

## Phase 1: LOAD

### 1.1 Read Work Order

```bash
cat $ARTIFACTS_DIR/work-order.md
```

Extract: PR number, cluster, phases, summary, files changed.

### 1.2 Read Progress

```bash
cat $ARTIFACTS_DIR/progress.md
```

Extract: completed phases, commit hashes, file lists.

### 1.3 Read Validation

```bash
cat $ARTIFACTS_DIR/validation.md
```

### 1.4 Get Branch Info

```bash
BRANCH=$(git branch --show-current)
COMMIT_COUNT=$(git rev-list --count origin/$BASE_BRANCH..HEAD)
echo "Branch: $BRANCH, Commits: $COMMIT_COUNT"
```

### 1.5 Check for Existing PR

```bash
gh pr list --head $(git branch --show-current) --json number,url,state 2>/dev/null
```

If PR already exists, update it instead of creating a new one.

---

## Phase 2: CREATE PR

### 2.1 Build PR Body

Use this template, filling in from the work order and progress artifacts:

```bash
gh pr create --draft --base $BASE_BRANCH --title "<title>" --body "$(cat <<'PREOF'
## Summary

Cleanup PR-{NN}: {summary from work order}

**Cluster**: C{N} — {cluster title}
**Phases**: {phase list with descriptions}
**Source**: `docs/audit/cleanup-plan.md`

## Changes

{For each phase:}
- **P{N}**: {description} ({files changed})

## Verification

- [x] TypeCheck passes (`pnpm exec nx run-many -t typecheck`)
- [x] Lint passes (`pnpm exec nx run-many -t lint`)
- [x] Related tests pass
- [x] Phase-specific verification commands pass

## Test Plan

- [ ] Verify no regressions in affected test suites
- [ ] Review diff against cleanup-plan.md phase descriptions
- [ ] Check that resolved decisions (D-XXX) are implemented correctly

## References

- Cleanup plan: `docs/audit/cleanup-plan.md` → PR-{NN}
{For each resolved decision:}
- Decision {D-XXX}: {one-line summary}

---
Generated by Archon workflow `execute-cleanup-pr`
PREOF
)"
```

### 2.2 Save PR Metadata

```bash
PR_NUMBER=$(gh pr view --json number -q .number)
PR_URL=$(gh pr view --json url -q .url)
echo "$PR_NUMBER" > $ARTIFACTS_DIR/.pr-number
echo "$PR_URL" > $ARTIFACTS_DIR/.pr-url
```

---

## Phase 3: OUTPUT

```markdown
## PR Created

**PR**: #{pr-number}
**URL**: {pr-url}
**Status**: Draft

Proceeding to parallel code review.
```
