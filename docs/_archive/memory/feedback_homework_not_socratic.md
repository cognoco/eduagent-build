Homework mode should explain and give examples, NOT use Socratic questioning.

**Why:** Socratic method on routine homework turns a 10-minute answer-check into a 40-minute interrogation. Students stop using the app. A teen at 9pm with 6 equations doesn't want "what's your first step?" — they want to know if x = 6 is correct, and if not, WHERE the error is with a brief explanation.

**How to apply:**
- Homework "Check my answer" mode: student gives answer → AI verifies right/wrong → if wrong, points to the specific error with a brief explanation. No back-and-forth.
- Homework "Help me" mode: AI explains the approach, shows a SIMILAR worked example, then lets the student try. Not Socratic — more like a teacher showing how to do a similar problem, then watching the student do theirs.
- Keep responses SHORT. Especially for teens. No multi-paragraph explanations when one sentence and an example will do.
- Socratic questioning stays for LEARNING mode (exploratory, curriculum-driven sessions). Homework is a different context — the student has a specific task with a deadline.
- This overrides FR31 ("NEVER provide direct answers") for homework. The AI still doesn't DO the homework — but it explains, gives examples, and verifies. There's a difference between "here's the answer" and "here's how to get the answer."
