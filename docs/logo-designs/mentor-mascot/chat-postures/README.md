# Octo Mate Chat Postures

Generated from the canonical `octo-mate.svg` path extraction via `apps/mobile/src/components/common/octo-mate-paths.ts`.

These are design assets only. The current mascot spec says the octopus is a ceremony character, not an everyday chat face. Wire these into chat only after that product ruling changes.

## Notes

- Transparent SVG and PNG outputs.
- The body, limbs, suckers, and beanie come from the canonical 105-path extraction.
- The expression layer is intentionally redrawn on top so each posture can read at chat-avatar size while preserving the mascot silhouette.
- The referenced `octopus 3D icons.png` informed the rounded icon treatments, but these outputs intentionally stay closer to the locked Octo Mate source than to a generic 3D redraw.
