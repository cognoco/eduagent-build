Language assessments should check usable language production: target-language words/chunks, English translations, spelling tolerance, and tiny exchanges.

**Why:** In the 2026-05-18 review flow, generic assessment prompts asked for "main ideas" about Greetings & Introductions and follow-ups drifted into "what is/when use a greeting" framing. That is not useful language education; learners already know what greetings are.

**How to apply:** For language reviews, ask concrete tasks like "Can you say hello in Italian?", "Add one more Italian greeting", "Translate one phrase into English", or "Use it in a two-line exchange." Avoid broad summaries, "main ideas", "what other words did we cover", and culture-ish questions unless the lesson explicitly taught culture/register.
