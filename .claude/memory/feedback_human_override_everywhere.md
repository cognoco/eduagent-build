type: feedback
---

Every AI-driven interaction must have a human override escape hatch.

**Why:** The AI is a guide, not an authority. If the LLM suggests subjects, the student must be able to type their own. If the system recommends a topic, the student must be able to choose differently. Removing human agency makes the tool feel controlling, not helpful.

**How to apply:**
- Subject selection: allow manual text input even when LLM suggests options
- Topic ordering: recommendations, not forced sequences
- Session flow: student can redirect, skip, or challenge any AI suggestion
- Coaching cards: suggestions, not mandates
- Audit every chat/session screen for missing override points
- This is a core product principle, not a per-feature decision
